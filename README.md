## 目录树

```
.
├─config      - 各种配置  用于项目中涉及的所以配置
├─controller  - 控制层 相当于 view
├─dao         - 数据库操作定义层 接口方法定义
│  └─impl     - 数据库操作实现层
├─db          - 数据库连接层 提供连接后的db
├─module      - 实体层
├─request     - 所有的请求体结构
├─response    - 所有的返回体结构
├─router      - 路由
├─service     - 业务逻辑定义层 接口方法定义
│  └─impl     - 业务逻辑实现层
├─sql         - 数据库设计及定义
├─test        - 测试
└─util        - 工具包
```
